Résumé de mon cour Azure Static Web App

👨‍💻 Étapes réalisées

  • ✅ Création d’un compte GitHub
  • ✅ Création d’un dépôt avec le modèle vanilla-basic
  • ✅ Connexion au portail Azure
  • ✅ Création d’une ressource Static Web App
  • ✅ Liaison entre GitHub et Azure
  • ✅ Déploiement automatique grâce à GitHub Actions
  • ✅ Modification de la page index.html
  • ✅ Test de mise à jour en ligne ✔️

📘 Ce que j'ai appris

Ce cours m’a appris à créer, héberger et mettre à jour un site web statique grâce à Azure et GitHub. Je peux désormais publier facilement une page web et la maintenir à jour sans intervention manuelle sur le serveur.
