Résumé de mon cours Azure

Mon parcours depuis le début

Depuis le début du cours, j’ai appris pas mal de choses sur Azure et comment déployer un site web dans le cloud. C'était un peu nouveau pour moi au début, mais étape par étape j'ai compris comment ça fonctionne.

J’ai commencé par créer une machine virtuelle (VM) sur Azure, que j’ai ensuite allumée et à laquelle je me suis connecté à distance. Dessus, j’ai installé un serveur web pour tester si je pouvais afficher une page depuis l’extérieur. Et ça a marché !

Ensuite, j’ai découvert GitHub. J’ai créé un compte, un dépôt avec une page web toute simple. Puis, j’ai relié ce dépôt à Azure Static Web Apps. Une fois que tout était connecté, j’ai pu déployer ma page web automatiquement.

Ce que j’ai trouvé super, c’est qu’il suffit de modifier le fichier HTML dans GitHub et ça met à jour le site automatiquement sur Azure. J’ai fait le test en changeant le texte de la page et j’ai bien vu le changement en ligne.

Ce que je retiens

Grâce à ce cours, j’ai appris à utiliser Azure, à créer une machine virtuelle, à me connecter dessus, à installer un serveur web, et aussi à utiliser GitHub pour déployer un site. Franchement, c’est motivant de voir son site en ligne comme ça.
