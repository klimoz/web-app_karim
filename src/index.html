Résumé de mon cours Azure & Déploiement Web

📅 Introduction du cours

Ce cours m’a permis de découvrir l’univers du cloud computing avec Azure et de comprendre les bases du déploiement d’applications web. Voici un résumé étape par étape de ce que j’ai appris et réalisé.

👨‍💻 Étapes réalisées depuis le début

  • ✅ Introduction à Microsoft Azure et découverte de son portail
  • ✅ Création d’une machine virtuelle (VM)
  • ✅ Connexion à la VM via le bureau à distance
  • ✅ Installation d’un serveur web (IIS ou Apache) sur la VM
  • ✅ Test de la page web sur la VM via l’adresse IP publique
  • ✅ Découverte de GitHub et création d’un compte
  • ✅ Création d’un dépôt GitHub à partir d’un template statique
  • ✅ Liaison entre GitHub et Azure via Static Web Apps
  • ✅ Déploiement automatique de la page grâce à GitHub Actions
  • ✅ Modification de la page HTML et test de la mise à jour automatique

📘 Ce que j’ai appris

Ce parcours m’a permis de comprendre les bases de :

  • 🔧 La gestion de ressources dans Azure
  • 📡 La connexion à distance et l’utilisation de VM
  • 🌐 Le déploiement d’un site web sur le cloud
  • ⚙️ L'automatisation des déploiements avec GitHub Actions
  • 🧠 L'importance des outils collaboratifs comme GitHub
